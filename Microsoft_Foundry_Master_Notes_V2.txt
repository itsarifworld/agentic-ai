SECTION 2: WHAT IS MICROSOFT FOUNDRY?
====================================================================

Microsoft Foundry is Microsoft's platform for building AI applications,
AI agents, and enterprise AI solutions.

It provides:

1. Model hosting
2. Agent development
3. Prompt management
4. Tool integration
5. Evaluation
6. Deployment
7. Monitoring

Think of Foundry as:

"Azure + OpenAI + Agent Framework + Deployment Platform"

SECTION 3: WHAT IS AN AGENT?
====================================================================

Normal LLM:

Input → Model → Output

Agent:

Input
 ↓
Reasoning
 ↓
Tool Usage
 ↓
External Data Access
 ↓
Final Answer

Agents can:

- Search the web
- Call APIs
- Access databases
- Maintain conversations
- Execute workflows

Example:

User:
"What was ENIAC?"

Agent:
1. Understands question
2. Searches knowledge/tool
3. Generates answer
4. Returns response

SECTION 18: COMMON REAL-WORLD ERRORS
====================================================================

ERROR 1

Command Palette Missing

Fix:

View → Command Palette

or

Cmd + Shift + P

------------------------------------------------

ERROR 2

Python Extension Missing

Symptoms:

Python commands unavailable.

Fix:

Install Python Extension by Microsoft.

------------------------------------------------

ERROR 3

az login not recognized

Fix:

Install Azure CLI.

------------------------------------------------

ERROR 4

401 Unauthorized

Cause:

Not logged into Azure.

Fix:

az login

------------------------------------------------

ERROR 5

403 Forbidden

Cause:

Wrong subscription.

Fix:

Verify Azure subscription.

------------------------------------------------

ERROR 6

405 Method Not Allowed

This is the exact issue you encountered.

Typical Causes:

1. Wrong endpoint
2. Wrong API version
3. Wrong base URL
4. Incorrect responses path

Debug Checklist:

- Verify endpoint copied correctly
- Verify agent published
- Verify API version
- Verify SDK version

------------------------------------------------

ERROR 7

Internal Error Communicating with Agent

Cause:

Underlying request failed.

Check:

- Internet
- Endpoint
- Authentication
- Azure login

SECTION 19: INTERVIEW QUESTIONS
====================================================================

Q1. What is Microsoft Foundry?

Answer:

A platform for building, deploying, evaluating,
and managing AI applications and agents.

------------------------------------------------

Q2. What is an AI Agent?

Answer:

An AI system capable of reasoning and using tools
to perform tasks beyond simple text generation.

------------------------------------------------

Q3. Why use Responses API?

Answer:

Provides unified support for conversational,
agentic, and tool-enabled AI applications.

------------------------------------------------

Q4. What is Entra ID Authentication?

Answer:

Microsoft identity-based authentication system
used to securely access Azure resources.

------------------------------------------------

Q5. Why use Virtual Environments?

Answer:

To isolate project dependencies and avoid conflicts.
